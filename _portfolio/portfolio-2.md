title: "MolProp-MLOps-Demo"
excerpt: "This project implements a production-style molecular property prediction system for estimating aqueous solubility directly from SMILES strings. The workflow includes reproducible model training on the Delaney ESOL dataset, RDKit-based molecular featurization, and scikit-learn regression models for property prediction. Beyond model development, the project includes command-line inference, a FastAPI deployment interface, a CustomTkinter desktop GUI, Docker-based containerization, continuous integration, automated tests, and model documentation. By combining cheminformatics, machine learning, and software engineering practices, the system demonstrates an end-to-end workflow for building, validating, and deploying molecular property prediction tools."
